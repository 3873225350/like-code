贪吃蛇 TUI 游戏
========================================

游戏简介:
  在终端中运行的经典贪吃蛇游戏。

操作说明:
  方向键 / WASD : 控制蛇的移动方向
  q             : 退出游戏
  r             : 重新开始游戏

运行方式:
  python3 snake_game.py
  或
  python3 snake_launcher.py

游戏规则:
  - 控制蛇吃食物，蛇身变长，分数增加
  - 撞到墙壁或自己身体则游戏结束
  - 每个食物 +10 分

  - mm7 (MiniMax-M2.7): 编写启动器和说明文档
